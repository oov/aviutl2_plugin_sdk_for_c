aviutl2_plugin_sdk_for_c
========================

このリポジトリは、AviUtl 用プラグイン開発キット aviutl2_plugin_sdk の C 言語バインディングを提供します。  
元々のヘッダーでも概ね変換せずに使える形で記述されていますが、一部については変換が必要です。

個人的な好みにより以下のような基準で変換しています。

- C 言語用に変換
- コメントは英語に翻訳
- 型の名前をスネークケースに変更
- 文字列型を標準的な型に置き換え
